Microgrid Simulation — Solar PV + Battery Storage + Resistive Load

Overview:
A microgrid simulation built in MATLAB Simulink and Simscape Electrical modeling the interaction between a solar photovoltaic source, battery storage, and a resistive load over a simulated day cycle.

What it models:
Solar PV generation represented as a sinusoidal current source simulating irradiance from sunrise to sunset
12V battery storage that charges during peak solar generation and discharges when solar output drops below load demand
10 ohm resistive load representing a constant power consumer
Voltage regulation demonstrating how battery storage stabilizes load voltage despite variable solar input

Tools Used:
MATLAB Simulink
Simscape Electrical
AutoCAD Electrical
